O que são Clients?

Voltar ao Índice

Client é um lado do programa que o usuário interage e usa para acessar a web. Um programa Client roda na máquina local fazendo uma requisição para o Server. Exemplos de client são: Computadores desktop, smartphones, laptop, etc.

O que são Servers

Como é uma estrutura básica de um HTML?

Voltar ao Índice

HTML é uma linguagem de marcação, ou seja, as tags HTML são marcadores que indicam para o navegador qual comportamento o texto dentro daquela tag deve ter. Para o navegador entender um documento HTML, ele deve ter uma estrutura básica:

  1. <!DOCTYPE html> É a tag que define que este documento se trata de um documento HTML5
  2. <html> Está é a base de todo o documento, é dentro dela que colocaremos todo os elementos do nosso site
  3. <head> As informações contidas dentro da head são para passar informações diretamente para o navegador, informações que podem mudar a apresentação da página html
  4. <title> Aqui colocamos o título da página que aparece na aba do navegador
  5. <body> Corpo da página, aqui fica todos os conteúdos visíveis para o usuário

Linguagem de programação: o que é?

*de acordo com o site Rockcontent

Voltar ao Índice

É uma linguagem formal que, através de uma série de instruções, permite que um programador escreva um conjunto de ordens, ações consecutivas, dados e algoritmos para criar programas que controlam o comportamento físico e lógico de uma máquina. Programador e máquina se comunicam por meio dessa linguagem, permitindo especificar, com precisão, aspectos como:

  • Quais dados um software deve operar;
  • Como esses dados devem ser armazenados ou transmitidos;
  • Quais ações o software deve executar, de acordo com cada circunstância variável.
